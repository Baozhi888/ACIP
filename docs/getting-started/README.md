## 开始使用

如果您是ACIP的新用户，我们建议您按照以下步骤开始：

1. 阅读[ACIP介绍](./introduction.md)了解核心概念
2. 按照[快速开始](./quick-start.md)指南设置您的环境
3. 探索[示例应用](https://github.com/acip-ai/examples)以获取灵感
